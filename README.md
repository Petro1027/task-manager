# Task Manager

Modern fullstack Kanban / task management webalkalmazás portfólió célra.

## Stack

- Frontend: React + TypeScript + Vite + Tailwind
- Backend: Node.js + Express + TypeScript
- Database: PostgreSQL
- ORM: Prisma
- Drag and drop: dnd-kit
- Validation: Zod
- Forms: react-hook-form
- Server state: TanStack Query
- Testing: Playwright
- Containerization: Docker + Docker Compose
- CI: GitHub Actions

## Planned features

- Register / login / logout
- Own boards
- 3-column Kanban view
- Task create / edit / delete
- Archive
- Priority / category / due date
- Search / filtering
- Drag and drop + order persistence
- HU / EN language switch
- Google Calendar link from task
- Dashboard statistics
